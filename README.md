# ACH Impact Intelligence Platform

## Setup

1. Run `supabase-schema.sql` in your Supabase SQL editor
2. Add environment variables in Vercel:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
3. Deploy

## Pages

- `/` - Landing page
- `/login` - Login
- `/partner/dashboard` - Employment partner dashboard
- `/partner/training` - Training partner dashboard
- `/partner/baseline` - Baseline role data
- `/partner/milestones` - Milestone reviews
- `/staff/dashboard` - Staff overview
- `/staff/candidates` - Manage candidates
- `/staff/partners` - Manage partners
